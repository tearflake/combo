# Specification

Specification for combo framework includes grammars and pseudocode for embraced programming paradigms:

* imp - simple imperative programming framework:  
  [grammar rules](./imp.rbnf), [pseudocode](./imp.pseudo)

* fnc - simple functional programming framework:  
  [grammar rules](./fnc.rbnf), [pseudocode](./fnc.pseudo)

* rwr - simple rewriting programming framework:  
  [grammar rules](./rwr.rbnf), [pseudocode](./rwr.pseudo)

* combo - an umbrella framework combining the above three programming paradigms:  
  [grammar rules](./combo.rbnf), [pseudocode](./combo.pseudo)

Although the pseudocode is short, it defines combo framework to the terse detail due to selected algorithms simplicity.
